## 🚀 5分钟开始

### 第1步：安装
```
VS Code扩展页面搜索"Caret" → 安装
```

### 第2步：API密钥设置  
```
在设置中输入要使用的AI提供商API密钥
(支持OpenAI, Claude, Gemini等36个提供商)
```

### 第3步：选择角色
```
在侧边栏选择喜欢的AI角色
或创建专属自定义角色
```

### 第4步：开始编程！
```
按Ctrl+'打开聊天窗口
输入"你好！帮我做个React组件"
```
