Cline은 Fireworks AI 플랫폼을 통해 모델 액세스를 지원하며, 이는 광범위한 최첨단 오픈 소스 언어 모델에 빠르고 비용 효율적인 액세스를 제공합니다. 속도와 안정성을 위해 구축된 Fireworks AI는 OpenAI 호환 API 및 최대 256,000 토큰의 컨텍스트 창을 갖춘 서버리스 배포 옵션을 제공합니다.

**웹사이트:** [https://fireworks.ai/](https://fireworks.ai/)

### API 키 얻기

1.  **가입/로그인:** [Fireworks AI](https://fireworks.ai/)로 이동하여 계정을 생성하거나 로그인합니다.
2.  **API 키로 이동:** 로그인 후 계정 설정의 [API 키 페이지](https://app.fireworks.ai/settings/users/api-keys)로 이동합니다.
3.  **키 생성:** "API 키 생성"을 클릭하고 키에 설명적인 이름(예: "Cline")을 지정합니다.
4.  **키 복사:** **중요:** API 키를 _즉시_ 복사하십시오. 다시 볼 수 없습니다. 안전하게 저장하십시오.

### 지원되는 모델

Cline은 다음 Fireworks AI 모델을 지원합니다.

-   `accounts/fireworks/models/kimi-k2-instruct` (기본값)
-   `accounts/fireworks/models/qwen3-235b-a22b-instruct-2507`
-   `accounts/fireworks/models/qwen3-coder-480b-a35b-instruct`
-   `accounts/fireworks/models/deepseek-r1-0528`
-   `accounts/fireworks/models/deepseek-v3`

**모델 세부 정보:**

| 모델 | 컨텍스트 창 | 최적 용도 | 가격 (1M 토큰당) |
|---|---|---|---|
| Kimi K2 | 128K | 일반 작업, 에이전트 기능 | 입력 \$0.60, 출력 \$2.50 |
| Qwen3 235B | 256K | 비용 효율적인 일반 사용 | 입력 \$0.22, 출력 \$0.88 |
| Qwen3 Coder | 256K | 코드 생성 및 디버깅 | 입력 \$0.45, 출력 \$1.80 |
| DeepSeek R1 | 160K | 복잡한 추론, 함수 호출 | 입력 \$3.00, 출력 \$8.00 |
| DeepSeek V3 | 128K | 강력한 일반 성능 | 입력 \$0.90, 출력 \$0.90 |

### Cline 구성

1.  **Cline 설정 열기:** Cline 패널에서 설정 아이콘(⚙️)을 클릭합니다.
2.  **공급자 선택:** "API 공급자" 드롭다운에서 "Fireworks AI"를 선택합니다.
3.  **API 키 입력:** Fireworks AI API 키를 "Fireworks AI API 키" 필드에 붙여넣습니다.
4.  **모델 선택:** "모델" 드롭다운에서 원하는 모델을 선택합니다. 기본 모델은 Kimi K2입니다.

### 팁 및 참고 사항

-   **비용 효율적:** Fireworks AI는 독점 모델보다 훨씬 저렴한 가격을 제공하면서 경쟁력 있는 성능을 유지합니다.
-   **대규모 컨텍스트 창:** 대부분의 모델은 128K-256K 토큰을 지원하여 대규모 문서 처리 및 확장된 대화 유지에 적합합니다.
-   **OpenAI 호환성:** 공급자는 스트리밍 지원 및 사용량 추적을 통해 OpenAI 호환 API 형식을 사용합니다.
-   **속도 제한:** Fireworks AI는 사용량 기반 속도 제한을 가집니다. 대시보드에서 사용량을 모니터링하고 필요한 경우 플랜 업그레이드를 고려하십시오.
-   **API 키:** 보안을 위해 로컬 머신에 저장됩니다.
-   **가격 책정:** 현재 요금은 [Fireworks AI 가격 책정 페이지](https://fireworks.ai/pricing)를 참조하십시오. 표시된 가격은 백만 토큰당입니다.
