Cline은 최첨단 AI 모델을 VS Code 에디터에 직접 가져오는 오픈 소스 AI 코딩 에이전트입니다. 자동 완성 도구와 달리, Cline은 전체 코드베이스를 이해하고, 복잡한 변경사항을 계획하며, 다단계 작업을 실행할 수 있는 진정한 코딩 에이전트입니다.

## 타협 없는 오픈 소스 AI 코딩

Cline은 제한 없이, 놀라움 없이, 모델 생태계 종속성 없이 최첨단 AI에 대한 직접적이고 투명한 접근을 제공합니다. 모든 결정을 확인하고, 원하는 모델을 선택하며, 비용을 통제하세요.

### 완전한 투명성

Cline이 파일을 읽고, 접근 방식을 고려하며, 변경사항을 제안하는 과정을 실시간으로 관찰하세요. 모든 결정이 보이고, 모든 편집은 실행 전에 검토할 수 있습니다. 이것은 단순한 "설명 가능한 AI"가 아니라 완전한 투명성입니다.

### 당신의 모델, 당신의 통제

복잡한 추론에는 Claude를, 대규모 컨텍스트에는 Gemini를, 효율성에는 Qwen3 Coder를 사용하세요. 새로운 모델이 출시되면 즉시 전환할 수 있습니다. 당신의 API 키, 당신의 선택. 혁신을 가로막지 않습니다.

### 실제 엔지니어링을 위한 구축

Cline은 다음을 수행할 수 있습니다:
-   전체 코드베이스에서 **파일 읽기 및 쓰기**
-   **터미널 명령 실행** 및 오류 디버깅
-   코드를 작성하기 전에 **복잡한 기능 계획**
-   MCP 서버를 통한 **외부 시스템 연결**
-   지능적인 컨텍스트 관리로 **대규모 코드베이스 이해**

## Plan & Act 모드

Cline은 코드베이스를 탐색하고 단 한 줄의 코드를 작성하기 전에 포괄적인 계획을 세우기 위해 함께 작업하여 프로젝트의 전체 컨텍스트를 이해합니다.

**Plan 모드**는 복잡한 작업을 위해 - Cline이 탐색하고, 질문하며, 상세한 구현 계획을 세웁니다.

**Act 모드**는 실행을 위해 - Cline이 완전한 투명성과 통제 하에 계획을 구현합니다.

## 설계상 제로 트러스트

당신의 코드는 저희 서버에 닿지 않습니다. Cline은 당신의 API 키로 완전히 클라이언트 측에서 실행되어, 엄격한 보안 요구사항을 가진 기업의 유일한 선택지가 됩니다.

**오픈 소스**는 보안팀이 모든 라인을 검토할 수 있다는 의미입니다. Cline이 어떻게 작동하는지, AI 제공자에게 무엇을 보내는지, 결정이 어떻게 내려지는지 정확히 확인하세요.

## 주요 기능

### Focus Chain
작업 전반에 걸쳐 실시간 진행 상황 추적이 포함된 자동 할 일 목록 관리. 긴 프로젝트에서 Cline이 궤도를 유지할 수 있도록 합니다.

### Auto Compact
대화가 길어지면, Cline이 자동으로 요약하여 컨텍스트를 보존하면서 작업을 계속할 수 있는 공간을 확보합니다.

### Deep Planning
복잡한 기능의 경우, Cline이 코드베이스를 조사하고, 명확화 질문을 하며, 포괄적인 구현 계획을 세웁니다.

### MCP Integration
Model Context Protocol을 통해 데이터베이스, API, 문서에 연결합니다. Cline이 모든 외부 시스템에 대한 가교 역할을 합니다.

### .clinerules
코딩 표준, 아키텍처 패턴 또는 팀 규칙을 포함하여 Cline이 따르는 프로젝트별 지침을 정의합니다.

## 개발자가 Cline을 선택하는 이유

**100% 오픈 소스** - GitHub의 모든 코드 라인. 코드를 읽고, 개선하고, 업무에 신뢰하는 개발자들로부터 48,000개 이상의 스타를 받았습니다.

**추론 게임 없음** - 저희는 AI 사용으로 수익을 얻지 않습니다. 다른 사람들이 컨텍스트를 제한하거나 더 저렴한 모델로 라우팅하는 동안, 저희는 모든 모델의 전체 기능에 대한 무제한 접근을 제공합니다.

**설계상 미래 증명** - 새로운 모델이 출시되었나요? 즉시 사용하세요. Cline은 모든 AI 제공자, 모든 모델과 작동합니다.

**진정한 가시성** - 읽은 모든 파일, 고려된 모든 결정, 사용된 모든 토큰을 확인하세요.
