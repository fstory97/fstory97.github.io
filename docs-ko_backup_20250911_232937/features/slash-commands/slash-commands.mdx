## 슬래시 명령이 중요한 이유

슬래시 명령은 다음을 통해 Cline과의 상호 작용을 향상시킵니다.

1.  **효율성**: 몇 번의 키 입력으로 복잡한 작업을 빠르게 시작합니다.
2.  **컨텍스트 인식**: 명령은 현재 작업 컨텍스트를 활용하여 관련성 높은 결과를 제공합니다.
3.  **워크플로 간소화**: 채팅 인터페이스를 벗어나지 않고도 강력한 기능을 직접 트리거합니다.
4.  **자동화**: 반복적인 설정 또는 정보 수집 단계를 자동화합니다.

## 시작하기

슬래시 명령을 사용하려면:

1.  채팅 입력 필드에 `/`를 입력합니다.
2.  사용 가능한 명령 목록에서 선택하거나 명령 이름을 계속 입력합니다.
3.  필요한 경우 추가 매개변수를 제공합니다.
4.  Enter를 눌러 명령을 실행합니다.

Cline은 명령을 처리하고 요청된 작업을 수행합니다.

## 내부 작동 방식

슬래시 명령을 사용하면 내부적으로 다음과 같은 일이 발생합니다.

1.  **감지**: 메시지를 보내면 Cline은 텍스트에서 `/command` 패턴을 감지합니다.
2.  **처리**: 감지된 각 명령에 대해 Cline은 다음을 수행합니다.
    -   명령 유형 및 매개변수 결정
    -   관련 내부 도구 또는 워크플로 트리거
    -   필요한 컨텍스트(예: 코드베이스, 터미널 출력) 수집
3.  **실행**: 명령이 실행되고 결과가 채팅 패널에 표시됩니다.

이 프로세스는 슬래시 명령을 사용할 때마다 자동으로 원활하게 발생하여 AI 지원에 대한 빠르고 직접적인 액세스를 제공합니다.
