### 基本工具

以下是开发所需的核心工具：

-   **Node.js & npm：** JavaScript 和网络开发必需
-   **Git：** 用于跟踪代码变化和与他人协作
-   **包管理器：** 让安装其他开发工具变得简单的工具
    -   Homebrew 适用于 macOS
    -   Chocolatey 适用于 Windows
    -   apt/yum 适用于 Linux

> **提示：** 这些工具是您开发者工具包的基础。正确安装它们将为您的成功奠定基础！

> **专业提示：** Cline 会在运行每个命令之前向您展示。您始终保持控制！

### 将会发生什么

Cline 将指导您完成以下步骤：

1. 为您的系统安装适当的包管理器
2. 使用包管理器安装 Node.js 和 Git
3. 在运行前向您显示确切的命令（您批准每个步骤！）
4. 验证每个安装是否成功

> **注意：** 对于某些安装，您可能需要输入计算机的密码。这是正常的！

### 为什么这些工具很重要

-   **Node.js & npm：**
    -   使用 React 或 Next.js 等框架构建网站
    -   运行 JavaScript 代码
    -   安装 JavaScript 包
-   **Git：**
    -   保存代码的不同版本
    -   与其他开发者协作
    -   备份您的工作
-   **包管理器：**
    -   快速安装和更新开发工具
    -   保持您的环境有组织且是最新的

### 注意事项

> **提示：** 安装过程是交互式的——Cline 将逐步指导您！

-   所有命令在运行前都会显示给您批准。
-   如果您遇到任何问题，Cline 会帮助解决。
-   某些步骤您可能需要输入计算机密码。

### 新程序员的额外提示

#### 理解终端

终端是一个应用程序，您可以在其中键入命令与计算机交互。

-   **macOS：** 通过在 Spotlight 中搜索"Terminal"来打开它。
-   **示例：**

```
$ open -a Terminal
```

#### 理解 VS Code 功能

-   **VS Code 中的终端：** 直接从 VS Code 内运行命令！
    -   转到 **View > Terminal** 或按 \`Ctrl + \`\`。
    -   示例：

```
$ node -v
v16.14.0
```

-   **文档视图：** 您编辑代码文件的地方。
    -   从左侧的资源管理器面板打开文件。
-   **问题部分：** 查看代码中的错误或警告。
    -   通过单击灯泡图标或 **View > Problems** 来访问它。

#### 常见功能

-   **命令行界面（CLI）：** 运行命令的强大工具。
-   **权限：** 您可能需要向某些命令授予权限——这保持您的系统安全。
