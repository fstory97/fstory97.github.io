> **提示：** 如果您完全是编程新手，请在每一步都花些时间。不用急——Cline 在这里指导您！

### 开始使用

在您开始编程之前，确保您准备好了这些必需品：

#### 1. **VS Code**

一个流行、免费且功能强大的代码编辑器。

-   [<u>下载 VS Code</u>](https://code.visualstudio.com/)

**推荐的 YouTube 教程：** [<u>如何安装 VS Code</u>](https://www.youtube.com/watch?v=MlIzFUI1QGA)

> **专业提示：** 将 VS Code 安装在您的 Applications 文件夹（macOS）或 Program Files（Windows）中，以便从停靠栏或开始菜单轻松访问。

#### 2. **整理您的项目**

在您的文档文件夹中创建一个名为 `Cline` 的专用文件夹，用于所有编程项目：

-   **macOS:** `/Users/[your-username]/Documents/Cline`
-   **Windows:** `C:\Users\[your-username]\Documents\Cline`

在您的 `Cline` 文件夹中，清晰地组织项目结构：

-   `Documents/Cline/workout-app` _（例如，用于健身跟踪应用）_
-   `Documents/Cline/portfolio-website` _（例如，展示您的工作）_

> **提示：** 从一开始就保持项目井然有序，将为您后续节省时间和困惑！

#### 3. **安装 Cline VS Code 扩展**

通过在 VS Code 中直接安装 Cline 扩展来增强您的编程工作流程：

-   Cline 扩展入门教程

**推荐的 YouTube 教程：** [<u>如何在 VS Code 中安装扩展</u>](https://www.youtube.com/watch?v=E7trgwZa-mk)

> **专业提示：** 安装后，重新加载 VS Code 以确保扩展正确激活。

#### 4. **基本开发工具**

高效编程所需的基本软件：

-   Homebrew（macOS）
-   Node.js
-   Git

[<u>遵循我们关于安装基本开发工具的详细指南，获得 Cline 的逐步帮助。</u>](https://docs.cline.bot/getting-started/installing-dev-essentials#installing-dev-essentials)

**手动安装推荐的 YouTube 教程：**

-   **对于 macOS：**
    -   [<u>在 Mac 上安装 Homebrew</u>](https://www.youtube.com/watch?v=hwGNgVbqasc)
    -   [<u>在 macOS 2024 上安装 Git</u>](https://www.youtube.com/watch?v=B4qsvQ5IqWk)
    -   [<u>在 Mac 上安装 Node.js（M1 | M2 | M3）</u>](https://www.youtube.com/watch?v=I8H4wolRFBk)
-   **对于 Windows：**
    -   [<u>在 Windows 10/11 上安装 Git（2024）</u>](https://www.youtube.com/watch?v=yjxv1HuRQy0)
    -   [<u>在 Windows 10/11 中安装 Node.js</u>](https://www.youtube.com/watch?v=uCgAuOYpJd0)

> **注意：** 如果在安装过程中遇到权限问题，请尝试以管理员身份运行您的终端或命令提示符。

您已经准备就绪！开始使用 **Cline** 更智能、更快速地编程吧。
