### 概述

为了帮助让 Cline 为每个人变得更好，我们收集使用数据，以帮助我们了解开发者如何使用我们的开源 AI 编码代理。这个反馈循环对于改进 Cline 的功能和用户体验至关重要。

我们使用 PostHog，一个开源分析平台，用于数据收集和分析。我们的遥测实施完全透明 - 您可以查看[源代码](https://github.com/cline/cline/blob/main/src/services/posthog/telemetry/TelemetryService.ts)，了解我们跟踪的确切内容。

### 跟踪政策

隐私是我们的优先事项。默认情况下，所有收集的数据都是匿名的。如果您使用 Cline 账户登录，您的遥测数据将与您的账户关联，以帮助我们改进产品，并在您遇到问题时提供更好的支持。您的代码、提示和对话内容始终保持私有，永远不会被收集。

### 我们跟踪的内容

我们收集基本使用数据，包括：

**任务交互：** 任务开始和结束的时间，对话流程（不包含内容）\
**模式和工具使用：** 计划/执行模式之间的切换，正在使用的工具\
**令牌使用：** 关于对话长度的基本指标以估算成本（不是令牌的实际内容）\
**系统上下文：** 操作系统类型和 VS Code 环境详细信息\
**用户界面活动：** 导航模式和功能使用

为了完全透明，您可以检查我们的[遥测实施](https://github.com/cline/cline/blob/main/src/services/posthog/telemetry/TelemetryService.ts)，以查看我们跟踪的确切事件。

### 如何选择退出

Cline 中的遥测完全可选：

-   当您更新或安装我们的 VS Code 扩展时，您会看到关于我们遥测的消息
-   您可以随时在设置中更改您的偏好

Cline 还尊重 VS Code 的全局遥测设置。如果您在 VS Code 级别禁用了遥测，Cline 的遥测也会自动被禁用。
