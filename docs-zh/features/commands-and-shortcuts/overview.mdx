### 什么是编辑器集成？

编辑器集成是让您能够在工作场所直接使用 Cline 的命令和快捷键。无需先切换到 Cline 面板，您可以选择代码、右键单击，然后立即将其发送给 Cline 寻求帮助。
这些集成出现在 VSCode 的不同位置：

-   编辑器上下文菜单（右键菜单）- "添加到 Cline"
-   终端上下文菜单 - "添加到 Cline"
-   源代码控制视图 - "生成提交消息"
-   键盘快捷键 - 各种 Cline 命令
-   快速修复选项（灯泡菜单）- "用 Cline 修复"、"用 Cline 解释"、"用 Cline 改进"

### 工作原理

当您使用这些命令时，Cline：

-   捕获相关上下文（选定的代码、文件路径、终端输出等）
-   聚焦 Cline 界面
-   使用捕获的上下文创建对话
-   在某些情况下，自动生成建议的提示

在后台，这些命令使用 VSCode 的扩展 API 来注册命令、访问编辑器状态并控制 VSCode 的界面。
