파일을 Cline으로 드래그 앤 드롭하는 것은 이미지, 코드 및 기타 파일을 대화에 추가하는 빠른 방법입니다.

<Frame>
	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/dran-n-drop.gif"
		alt="파일을 Cline 채팅으로 드래그 앤 드롭"
	/>
</Frame>

<Note>VS Code의 특성상 파일을 Cline 채팅 입력으로 드래그 앤 드롭하려면 드래그하는 동안 `Shift` 키를 눌러야 합니다.</Note>

작업 공간 파일을 Cline으로 드래그 앤 드롭하면 자동으로 [파일 멘션](/features/at-mentions/file-mentions)이 생성됩니다. 이를 통해 경로를 입력할 필요 없이 대화에서 파일을 참조할 수 있습니다.

### Finder/파일 탐색기에서 드래그

시스템의 파일 관리자에서 Cline으로 파일을 직접 드래그할 수 있습니다.

	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/drag-n-drop-finder.gif"

### 지원되는 파일 형식

Cline은 외부 이미지, PDF, CSV, Excel 및 기타 텍스트 파일을 파일 시스템에서 드래그하는 것을 지원하며, 작업 공간의 파일도 지원합니다.
