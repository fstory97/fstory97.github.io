### 필수 도구

다음은 개발에 필요한 핵심 도구입니다.

-   **Node.js & npm:** JavaScript 및 웹 개발에 필요합니다.
-   **Git:** 코드 변경 사항을 추적하고 다른 사람들과 협업하는 데 사용됩니다.
-   **패키지 관리자:** 다른 개발 도구를 쉽게 설치할 수 있도록 하는 도구입니다.
    -   macOS용 Homebrew
    -   Windows용 Chocolatey
    -   Linux용 apt/yum

> **팁:** 이 도구들은 개발자 도구 키트의 기반입니다. 제대로 설치하면 성공적인 시작을 할 수 있습니다!

### Cline이 모든 것을 설치하도록 허용

운영 체제에 따라 다음 프롬프트 중 하나를 복사하여 **Cline**에 붙여넣으십시오.

#### macOS용

```
안녕 Cline! Mac을 소프트웨어 개발용으로 설정하는 데 도움이 필요해. Homebrew, Node.js, Git과 같이 코딩에 일반적으로 필요한 필수 개발 도구 및 기타 핵심 유틸리티를 설치하는 데 도움을 줄 수 있니? 단계별로 과정을 안내해 주면 좋겠어.
```

> **전문가 팁:** Cline은 각 명령을 실행하기 전에 보여줄 것입니다. 당신은 항상 제어권을 유지합니다!

### 어떤 일이 일어날까요?

Cline은 다음 단계를 안내할 것입니다.

1.  시스템에 적합한 패키지 관리자 설치
2.  패키지 관리자를 사용하여 Node.js 및 Git 설치
3.  실행하기 전에 정확한 명령 보여주기(각 단계를 승인합니다!)
4.  각 설치가 성공했는지 확인

> **참고:** 일부 설치에는 컴퓨터 암호를 입력해야 할 수 있습니다. 이는 정상입니다!

### 이 도구들이 중요한 이유

-   **Node.js & npm:**
    -   React 또는 Next.js와 같은 프레임워크로 웹사이트 구축
    -   JavaScript 코드 실행
    -   JavaScript 패키지 설치
-   **Git:**
    -   코드의 다른 버전 저장
    -   다른 개발자와 협업
    -   작업 백업
-   **패키지 관리자:**
    -   개발 도구를 빠르게 설치 및 업데이트
    -   환경을 체계적으로 유지하고 최신 상태로 유지

### 참고 사항

> **팁:** 설치 프로세스는 대화형입니다. Cline이 단계별로 안내해 드릴 것입니다!

-   모든 명령은 실행되기 전에 승인을 위해 표시됩니다.
-   문제가 발생하면 Cline이 문제 해결을 도와줄 것입니다.
-   특정 단계에서는 컴퓨터 암호를 입력해야 할 수 있습니다.

### 새로운 코더를 위한 추가 팁

#### 터미널 이해

터미널은 컴퓨터와 상호 작용하기 위해 명령을 입력할 수 있는 애플리케이션입니다.

-   **macOS:** Spotlight에서 "터미널"을 검색하여 엽니다.
-   **예시:**

```
$ open -a Terminal
```

#### VS Code 기능 이해

-   **VS Code의 터미널:** VS Code 내에서 직접 명령 실행!
    -   **보기 > 터미널**로 이동하거나 \`Ctrl + \`\`를 누릅니다.
    -   **예시:**

```
$ node -v
v16.14.0
```

-   **문서 보기:** 코드 파일을 편집하는 곳입니다.
    -   왼쪽의 탐색기 패널에서 파일을 엽니다.
-   **문제 섹션:** 코드의 오류 또는 경고를 봅니다.
    -   전구 아이콘을 클릭하거나 **보기 > 문제**로 액세스합니다.

#### 일반적인 기능

-   **명령줄 인터페이스(CLI):** 명령을 실행하기 위한 강력한 도구입니다.
-   **권한:** 특정 명령에 대한 권한을 부여해야 할 수 있습니다. 이는 시스템을 안전하게 유지합니다.
