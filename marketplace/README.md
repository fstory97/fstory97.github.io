## ✨ What is Caret?

Caret is the ultimate autonomous AI coding assistant that **actually understands your codebase**. Unlike simple autocomplete tools, Caret can read, write, and refactor entire codebases while following your project's conventions and best practices.

### 🚀 Key Features

#### 🤖 **Autonomous Code Generation**
- **Full file creation and editing** - Not just snippets, but complete implementations
- **Multi-file refactoring** - Seamlessly update related files across your project
- **Context-aware suggestions** - Understands your existing code patterns and conventions

#### 🧠 **Advanced AI Models**
- **Claude 3.5 Sonnet** - Industry-leading reasoning and code quality
- **GPT-4o & o1** - Latest OpenAI models for complex problem-solving  
- **Gemini Pro** - Google's most capable model for code understanding
- **20+ AI providers** - Choose the best model for each task

#### 🔧 **Powerful Tool Integration**
- **Terminal commands** - Execute bash, npm, git commands directly
- **Browser automation** - Web scraping and testing capabilities
- **File operations** - Read, write, search across your entire project
- **Git integration** - Smart commit messages and branch management

#### 🎨 **Professional UI/UX**
- **Clean, modern interface** - Intuitive chat-based interaction
- **Diff viewer** - See exactly what changes before applying
- **Project explorer** - Navigate and understand large codebases
- **Customizable themes** - Match your VS Code setup

## 🚀 Quick Start

### 1. Install from VS Code Marketplace
```bash
code --install-extension caret.caret
```

### 2. Configure Your AI Provider
1. Open Command Palette (`Ctrl+Shift+P`)
2. Run `Caret: Open Settings`
3. Add your API key for preferred AI provider
4. Select your default model

### 3. Start Coding!
1. Open any project in VS Code
2. Click the Caret icon in the sidebar
3. Ask Caret to help with your code
4. Watch the magic happen! ✨
