# 작업 관리

Cline을 사용하면 시간이 지남에 따라 많은 작업을 축적하게 됩니다. 작업 관리 시스템은 작업 공간을 효율적으로 유지하기 위해 작업 기록을 구성하고, 필터링하고, 검색하고, 정리하는 데 도움이 됩니다.

## 작업 기록 액세스

작업 기록은 다음을 통해 액세스할 수 있습니다.

1.  Cline 사이드바에서 "기록" 버튼 클릭
2.  명령 팔레트를 사용하여 "Cline: 작업 기록 표시" 검색

## 작업 기록 기능

작업 기록 보기는 몇 가지 강력한 기능을 제공합니다.

### 검색 및 필터링

-   **검색 창**: 상단의 퍼지 검색을 사용하여 내용으로 작업을 빠르게 찾습니다.
-   **정렬 옵션**: 다음 기준으로 작업을 정렬합니다.
    -   최신순 (기본값)
    -   오래된순
    -   가장 비싼 (가장 높은 API 비용)
    -   가장 많은 토큰 (가장 높은 토큰 사용량)
    -   가장 관련성 높은 (검색 시)
-   **즐겨찾기 필터**: 즐겨찾기 작업만 표시하도록 전환합니다.

### 작업 작업

기록 보기의 각 작업에는 여러 가지 사용 가능한 작업이 있습니다.

-   **열기**: 작업을 클릭하여 Cline 채팅에서 다시 엽니다.
-   **즐겨찾기**: 별 아이콘을 클릭하여 작업을 즐겨찾기로 표시합니다.
-   **삭제**: 개별 작업을 제거합니다(즐겨찾기는 삭제로부터 보호됩니다).
-   **내보내기**: 작업의 대화를 마크다운으로 내보냅니다.

## ⭐ 작업 즐겨찾기

즐겨찾기 기능은 보존하고 빠르게 찾고 싶은 중요한 작업을 표시할 수 있도록 합니다.

### 즐겨찾기 작동 방식

-   **즐겨찾기 표시**: 모든 작업 옆의 별 아이콘을 클릭하여 즐겨찾기 상태를 전환합니다.
-   **보호**: 즐겨찾기 작업은 개별 및 대량 삭제 작업으로부터 보호됩니다(재정의 가능).
-   **필터링**: 즐겨찾기 필터를 사용하여 중요한 작업에 빠르게 액세스합니다.

## 일괄 작업

작업 기록 보기는 여러 일괄 작업을 지원합니다.

-   **여러 개 선택**: 확인란을 사용하여 여러 작업을 선택합니다.
-   **모두 선택/선택 해제**: 모든 작업을 빠르게 선택하거나 선택 해제합니다.
-   **선택한 항목 삭제**: 선택한 모든 작업을 제거합니다.
-   **모두 삭제**: 기록에서 모든 작업을 제거합니다(즐겨찾기는 포함하도록 선택하지 않는 한 보존됩니다).

## 모범 사례

1.  **중요한 작업 즐겨찾기**: 참조 작업 또는 자주 액세스하는 대화를 즐겨찾기로 표시합니다.
2.  **정기적인 정리**: 성능 향상을 위해 오래되거나 사용하지 않는 작업을 주기적으로 제거합니다.
3.  **검색 사용**: 퍼지 검색을 활용하여 특정 대화를 빠르게 찾습니다.
4.  **가치 있는 작업 내보내기**: 외부 참조를 위해 중요한 작업을 마크다운으로 내보냅니다.

작업 관리는 Cline을 사용할 때 체계적인 워크플로를 유지하는 데 도움이 되며, 과거 대화를 빠르게 찾고, 중요한 작업을 보존하고, 기록을 깔끔하고 효율적으로 유지할 수 있도록 합니다.
