## スラッシュコマンドの仕組み

スラッシュコマンドは以下の方法で Cline に特別な指示を送信する簡単な方法です。

1.  **認識**: チャット入力フィールドで特別なコマンドを入力します
2.  **コンテキスト収集**: コマンドは現在のコンテキストを分析し適切な応答をする準備をします
3.  **ガイド付きインタラクション**: あなたに明確化の質問をしたり、プロセスを通じてガイドしたりします
4.  **実行**: 要求されたタスクまたはワークフローを実行します

## 使用方法

スラッシュコマンドを使用するには：

1.  チャット入力フィールドに `/` から始めて入力します
2.  特定のコマンド名を入力するか、利用可能なオプションから選択します
3.  問題について質問されたら回答します
4.  Enter を押してコマンドを実行します

Cline がコマンドを処理し、適切なアクションを実行します。

## メリット

スラッシュコマンドを使用すると以下のメリットがあります：

1.  **効率性**: 共通のタスクに対して Cline はコンテキストなしで `/command` 形式を認識します
2.  **一貫性**: 各タスクに対して Cline が適切な応答をします
    -   コマンドに固有の最適化された実行
    -   最良の結果のための標準化された手順
    -   問題に特化した洞察とガイダンス
3.  **実行**: 複雑なタスクに対する詳細で段階的なガイダンス

これらの機能はスラッシュコマンドを使用することで、より効率的でAIアシスタントとしてのより良い体験を提供します。
