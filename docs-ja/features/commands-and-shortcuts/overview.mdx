ClineはVSCodeのインターフェースに直接統合され、ワークフローを中断することなくAIアシスタンスにアクセスできます。これらの統合は、エディター全体のコンテキストメニュー、キーボードショートカット、クイック修正としてコマンドが表示されます。

### エディター統合とは何ですか？

エディター統合は、作業中の場所でClineを直接使用できるコマンドとショートカットです。最初にClineパネルに切り替える代わりに、コードを選択し、右クリックして、すぐにClineにヘルプを求めることができます。
これらの統合は、VSCode全体のさまざまな場所に表示されます：

-   エディターのコンテキストメニュー（右クリックメニュー）- 「Clineに追加」
-   ターミナルのコンテキストメニュー - 「Clineに追加」
-   ソース管理ビュー - 「コミットメッセージを生成」
-   キーボードショートカットとして - さまざまなClineコマンド
-   クイック修正オプション（電球メニュー）- 「Clineで修正」、「Clineで説明」、「Clineで改善」

### 動作の仕組み

これらのコマンドを使用すると、Clineは以下を実行します：

-   関連するコンテキスト（選択されたコード、ファイルパス、ターミナル出力など）をキャプチャ
-   Clineインターフェースにフォーカス
-   キャプチャしたコンテキストで会話を作成
-   場合によっては、提案されたプロンプトを自動生成

舞台裏では、これらのコマンドはVSCodeの拡張機能APIを使用してコマンドを登録し、エディターの状態にアクセスし、VSCodeのインターフェースを制御します。
