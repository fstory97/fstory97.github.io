Claude Code プロバイダーを使用すると、既存の Claude サブスクリプションを Cline で使用できます。Claude Max または Pro をお持ちの場合、追加の API コストを支払うことなく、Cline で Claude を使用できます。

## セットアップ

まず、システムに Claude Code をインストールして認証する必要があります：

1. **Claude Code をインストール**: Anthropic の[公式セットアップガイド](https://docs.anthropic.com/en/docs/claude-code/setup)に従って、Claude CLI をインストールし、認証してください。

2. **Cline で設定**:
    - Cline 設定を開く（⚙️ アイコン）
    - **API Provider** ドロップダウンから **Claude Code** を選択
    - Claude CLI 実行可能ファイルのパスを設定（通常は PATH にある場合は `claude` のみ）

### Claude Code パスの確認

Claude Code がどこにインストールされているか分からない場合：

-   **macOS / Linux**: ターミナルで `which claude` を実行
-   **Windows (コマンドプロンプト)**: `where claude` を実行
-   **Windows (PowerShell)**: `Get-Command claude` を実行

## サポートされているモデル

Claude Code プロバイダーは以下のモデルをサポートします：

-   `claude-sonnet-4-20250514`（推奨）
-   `claude-opus-4-1-20250805`
-   `claude-opus-4-20250514`
-   `claude-3-7-sonnet-20250219`
-   `claude-3-5-sonnet-20241022`
-   `claude-3-5-haiku-20241022`

## 仕組み

Claude Code を Cline で使用する場合の背景での処理：

Cline は Claude Code CLI をラップしてリクエストを処理します。メッセージを送信するたびに、Cline は新しい `claude` プロセスを開始し、会話を送信し、レスポンスをストリーミングで返します。AI 推論は Claude Code から来ますが、実際のファイル編集、ターミナルコマンド、その他のツールはすべて Cline によって処理されます。

主な違いは、他のプロバイダーのようにレスポンスが文字単位でストリーミングされないことです。代わりに、Claude Code は完全なリクエストを処理してから、完全なレスポンスを送り返します。

## 制限事項

Claude Code を使用する際に覚えておくべき点がいくつかあります：

-   メッセージ内の画像は、Claude Code が CLI を通じた画像アップロードをサポートしていないため、テキストプレースホルダーに変換されます
-   このプロバイダーではプロンプトキャッシュは利用できません
-   他のプロバイダーのようにレスポンスがリアルタイムでストリーミングされません

## トラブルシューティング

問題が発生した場合：

**認証の問題**: サブスクリプションアカウントで Claude Code にログインしていることを確認してください。`claude auth status` を実行して確認します。

**パスの問題**: Cline 設定の Claude CLI パスが正しいことを再確認してください。ターミナルで `claude --version` を実行して動作することを確認します。

**まだ問題がありますか?** この統合を積極的に改善しています。[GitHub](https://github.com/cline/cline/issues) で問題を報告するか、[Discord](https://discord.gg/cline) でヘルプを求めてください。

## サブスクリプションでの使用

Claude Max サブスクリプションをお持ちの場合、追加の API コストを支払っていないため、Cline での使用量は請求インターフェースで $0.00 と表示されます。使用量はサブスクリプション制限に対してカウントされますが、トークン単位の料金は表示されません。

サブスクリプションでの Claude Code の使用に関する詳細については、Anthropic のドキュメントを確認してください：

-   [Claude Code セットアップガイド](https://docs.anthropic.com/en/docs/claude-code/setup)
-   [Pro/Max プランでの Claude Code の使用](https://support.anthropic.com/en/articles/11145838-using-claude-code-with-your-pro-or-max-plan)
