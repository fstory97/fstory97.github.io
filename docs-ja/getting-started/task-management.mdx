# タスク管理

Clineを使用していると、時間の経過とともに多くのタスクが蓄積されます。タスク管理システムは、ワークスペースを効率的に保つために、タスク履歴を整理、フィルター、検索、クリーンアップするのに役立ちます。

## タスク履歴へのアクセス

タスク履歴には以下の方法でアクセスできます：

1. Clineサイドバーの「履歴」ボタンをクリック
2. コマンドパレットを使用して「Cline: Show Task History」を検索

## タスク履歴の機能

タスク履歴ビューは、いくつかの強力な機能を提供します：

### 検索とフィルタリング

-   **検索バー**: 上部のファジー検索を使用してコンテンツでタスクを素早く見つける
-   **ソートオプション**: タスクを以下でソート：
    -   最新（デフォルト）
    -   最古
    -   最も高価（最高のAPIコスト）
    -   最多トークン（最高のトークン使用量）
    -   最も関連性が高い（検索時）
-   **お気に入りフィルター**: お気に入りタスクのみを表示するトグル

### タスクアクション

履歴ビューの各タスクには、利用可能ないくつかのアクションがあります：

-   **開く**: タスクをクリックしてClineチャットで再度開く
-   **お気に入り**: スターアイコンをクリックしてタスクをお気に入りとしてマーク
-   **削除**: 個別のタスクを削除（お気に入りは削除から保護されています）
-   **エクスポート**: タスクの会話をMarkdownにエクスポート

## ⭐ タスクのお気に入り

お気に入り機能を使用すると、保存して素早く見つけたい重要なタスクをマークできます。

### お気に入りの仕組み

-   **お気に入りのマーク**: 任意のタスクの横にあるスターアイコンをクリックしてお気に入りステータスを切り替え
-   **保護**: お気に入りタスクは個別および一括削除操作から保護されています（オーバーライド可能）
-   **フィルタリング**: お気に入りフィルターを使用して重要なタスクに素早くアクセス

## 一括操作

タスク履歴ビューは、いくつかの一括操作をサポートします：

-   **複数選択**: チェックボックスを使用して複数のタスクを選択
-   **すべて選択/選択解除**: すべてのタスクを素早く選択または選択解除
-   **選択済みを削除**: 選択されたすべてのタスクを削除
-   **すべて削除**: 履歴からすべてのタスクを削除（お気に入りを含めることを選択しない限り、お気に入りは保持されます）

## ベストプラクティス

1. **重要なタスクをお気に入りに**: 参照タスクまたは頻繁にアクセスする会話をお気に入りとしてマーク
2. **定期的なクリーンアップ**: パフォーマンスを向上させるために、古いまたは使用されていないタスクを定期的に削除
3. **検索を使用**: ファジー検索を活用して特定の会話を素早く見つける
4. **価値のあるタスクをエクスポート**: 外部参照のために重要なタスクをMarkdownにエクスポート

タスク管理は、Clineを使用する際に整理されたワークフローを維持し、過去の会話を素早く見つけ、重要な作業を保存し、履歴をクリーンで効率的に保つのに役立ちます。
