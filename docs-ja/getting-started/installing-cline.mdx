### インストールオプション

-   **VS Code Marketplace（推奨）:** 標準のVS CodeとCursorユーザーにとって最速の方法。
-   **Open VSX Registry:** VSCodiumなどのVS Code互換エディター用。

### VS Code Marketplace: ステップバイステップセットアップ

Clineを起動して実行するには、以下の手順に従ってください：

1.  **VS Codeを開く:** VS Codeアプリケーションを起動します。

    > **注意:** VS Codeが「拡張機能を実行する可能性があります...」と表示した場合、「許可」をクリックしてください。

2.  **Clineフォルダを開く:** VS Codeで、Documentsに作成したClineフォルダを開きます。
3.  **拡張機能に移動:** VS Codeサイドのアクティビティバーにある拡張機能アイコンをクリックします（`Ctrl + Shift + X`または`Cmd + Shift + X`）。
4.  **'Cline'を検索:** 拡張機能の検索バーに`Cline`と入力します。

1. **拡張機能をインストール:** Cline拡張機能の横にある「インストール」ボタンをクリックします。
2. **Clineを開く:**
    - アクティビティバーのClineアイコンをクリックします。
    - または、コマンドパレット（`Ctrl/Cmd + Shift + P`）を使用して「Cline: Open In New Tab」と入力してより良いビューを得ます。
3. **トラブルシューティング:** Clineアイコンが表示されない場合は、VS Codeを再起動してみてください。

> **プロチップ:** VS CodeエディターにClineチャットウィンドウが表示されるはずです！

### Open VSX Registry

MarketplaceにアクセスできないVS Code互換エディター（VSCodiumやWindsurfなど）の場合：

1. エディターを開きます。
2. 拡張機能ビューにアクセスします。
3. 「Cline」を検索します。
4. saoudrizwanによる「Cline」を選択し、**インストール**をクリックします。
5. プロンプトが表示されたら再読み込みします。

### Clineアカウントの作成

Clineがインストールされたので、アカウントを設定しましょう：

1. **Clineにサインイン:**
    - Cline拡張機能の**サインイン**ボタンをクリックします。
    - [app.cline.bot](https://app.cline.bot)に移動してアカウントを作成します。
2. **無料クレジットで開始:**
    - クレジットカードは不要です！
3. **利用可能なAIモデル:**
    - Anthropic Claude 3.5-Sonnet（コーディングに推奨）
    - DeepSeek Chat（コスト効率的な代替案）
    - Google Gemini 2.0 Flash
    - その他 — すべてClineアカウント経由で利用可能。

### Clineとの最初の対話

構築を始める準備ができました！このプロンプトをClineチャットウィンドウにコピー＆ペーストしてください：

```
こんにちは、Cline！私のClineディレクトリに「hello-world」という名前の新しいプロジェクトフォルダを作成し、「Hello World」という大きな青いテキストが表示されるシンプルなウェブページを作成してもらえませんか？
```

> **プロチップ:** Clineがプロジェクトフォルダを作成し、最初のウェブページをセットアップしてくれます！

### Clineとの作業のヒント

-   **質問をする:** 何かわからないことがあれば、Clineに聞いてください！
-   **スクリーンショットを使用:** Clineは画像を理解できます — 作業中の内容を見せてください。
-   **エラーをコピー＆ペースト:** 解決策のためにエラーメッセージをチャットで共有してください。
-   **平易に話す:** 自分の言葉を使用してください — Clineがそれをコードに変換します。
