### はじめに

コーディングに飛び込む前に、これらの必需品が準備されていることを確認してください：

#### 1. **VS Code**

人気のある、無料で強力なコードエディター。

-   [<u>VS Codeをダウンロード</u>](https://code.visualstudio.com/)

**推奨YouTubeチュートリアル:** [<u>VS Codeのインストール方法</u>](https://www.youtube.com/watch?v=MlIzFUI1QGA)

> **プロチップ:** ドックやスタートメニューから簡単にアクセスできるよう、VS CodeをApplicationsフォルダ（macOS）またはProgram Files（Windows）にインストールしてください。

#### 2. **プロジェクトを整理する**

すべてのコーディングプロジェクト用にDocumentsフォルダに`Cline`という名前の専用フォルダを作成します：

-   **macOS:** `/Users/[your-username]/Documents/Cline`
-   **Windows:** `C:\Users\[your-username]\Documents\Cline`

`Cline`フォルダ内で、プロジェクトを明確に構造化します：

-   `Documents/Cline/workout-app` _(例：フィットネス追跡アプリ用)_
-   `Documents/Cline/portfolio-website` _(例：作品を紹介するため)_

> **ヒント:** 最初からプロジェクトを整理しておくことで、後で時間と混乱を節約できます！

#### 3. **Cline VS Code拡張機能をインストール**

VS Code内でCline拡張機能を直接インストールして、コーディングワークフローを強化します：

-   Cline拡張機能チュートリアルをはじめる

**推奨YouTubeチュートリアル:** [<u>VS Codeで拡張機能をインストールする方法</u>](https://www.youtube.com/watch?v=E7trgwZa-mk)

> **プロチップ:** インストール後、拡張機能が適切にアクティベートされるようにVS Codeを再読み込みしてください。

#### 4. **必須開発ツール**

効率的なコーディングに必要な基本ソフトウェア：

-   Homebrew（macOS）
-   Node.js
-   Git

[<u>Clineからのステップバイステップヘルプで必須開発ツールのインストールに関する詳細ガイドに従ってください。</u>](https://docs.cline.bot/getting-started/installing-dev-essentials#installing-dev-essentials)

**手動インストール用の推奨YouTubeチュートリアル:**

-   **macOS用:**
    -   [<u>MacでHomebrewをインストール</u>](https://www.youtube.com/watch?v=hwGNgVbqasc)
    -   [<u>macOS 2024でGitをインストール</u>](https://www.youtube.com/watch?v=B4qsvQ5IqWk)
    -   [<u>MacでNode.jsをインストール（M1 | M2 | M3）</u>](https://www.youtube.com/watch?v=I8H4wolRFBk)
-   **Windows用:**
    -   [<u>Windows 10/11でGitをインストール（2024）</u>](https://www.youtube.com/watch?v=yjxv1HuRQy0)
    -   [<u>Windows 10/11でNode.jsをインストール</u>](https://www.youtube.com/watch?v=uCgAuOYpJd0)

> **注意:** インストール中に権限の問題が発生した場合は、ターミナルまたはコマンドプロンプトを管理者として実行してみてください。

すべて準備完了です！**Cline**でより賢く、より速くコーディングを始めましょう。
